# Projeto Casa 301 m²

Repositório para organizar o processo de concepção, decisão e documentação da casa de 301 m².

## Estrutura

- `/docs/sistema` – “Sisteminha” (questionário em etapas) para capturar preferências e direcionar o projeto.
- `/docs/persona` – Persona do projeto (perfil da casa, do casal, prioridades).
- `/docs/projeto` – Resumos e evolução do partido arquitetônico.
- `/docs/decisoes` – Registro de decisões sobre ambientes, acabamentos e tecnologias.
- `/data` – Arquivos JSON com respostas e preferências.
- `/drive_links` – Arquivos com links para pastas do Google Drive (fotos, referências, plantas, etc).

## Objetivo

Centralizar:
- Questionário de necessidades e gostos.
- Histórico de decisões.
- Links para materiais pesados no Google Drive.
- Geração de relatórios com ajuda de IA.
