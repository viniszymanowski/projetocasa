# Código de Obras - Resumo Técnico
## 1. Estrutura Geral
- Aplicação: Construções residenciais unifamiliares.
- Autoridade responsável: Secretaria Municipal de Obras e Urbanismo.

## 2. Normas de Aprovação
- Exigência de projeto arquitetônico, estrutural, elétrico e hidrossanitário.
- Responsabilidade técnica (ART/RRT).
- Documentos obrigatórios: planta de situação, cortes, fachadas, memorial descritivo.

## 3. Padrões Construtivos
- Pé-direito mínimo: 2,60 m em áreas habitáveis.
- Recuos mínimos: frente 5 m, fundos 2 m, laterais 1,5 m (verificar zoneamento).
- Área de ventilação: 1/8 da área do piso.
- Revestimentos: obrigatórios em áreas molhadas.

## 4. Licenciamento
- Etapas: Consulta de viabilidade → Aprovação de projeto → Alvará → Habite-se.
- Prazo médio de aprovação: 30 dias úteis (com documentação completa).
