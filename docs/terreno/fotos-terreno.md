# Fotos do Terreno

Estas são as fotos de referência do terreno. Este é apenas um exemplo inicial – depois vocês podem substituir por fotos reais tiradas no local.

![Terreno – vista modelo](/projetocasa/data/Fotos/terreno-modelo.jpg)

---

**Instruções de uso:**
- Quando tiver fotos reais do terreno, substitua o arquivo `terreno-modelo.jpg` por suas imagens.
- As novas fotos devem ser colocadas na pasta:  
  `/data/Fotos/`
- Você pode inserir várias imagens seguindo este formato:

```markdown
![Vista frontal](/projetocasa/data/Fotos/terreno-frente.jpg)
![Vista lateral](/projetocasa/data/Fotos/terreno-lateral.jpg)
![Vista aérea](/projetocasa/data/Fotos/terreno-aerea.jpg)
