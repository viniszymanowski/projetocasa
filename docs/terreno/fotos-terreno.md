**Instruções:**
- Quando tiver fotos reais, coloque-as em `/data/Fotos/`
- Use o mesmo formato acima, trocando o nome do arquivo e, se quiser, o `width` (largura) para ajustar a exibição.
- Exemplo:

```html
<img src="https://viniszymanowski.github.io/projetocasa/data/Fotos/terreno-frente.jpg" alt="Vista frontal" width="300px">
<img src="https://viniszymanowski.github.io/projetocasa/data/Fotos/terreno-lateral.jpg" alt="Vista lateral" width="300px">
